zmq_unbind(3)
==============


NAME
----
zmq_unbind - Another name for zmq_disconnect


SYNOPSIS
--------
int zmq_unbind (void '*socket', const char '*endpoint');


DESCRIPTION
-----------
The _zmq_unbind()_ has the same exact behavior as _zmq_disconnect()_.
Refer to linkzmq:zmq_disconnect[3].
